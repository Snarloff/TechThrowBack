## Descrição

Este é mais um projeto do repositório TechThrowback. Aqui, você encontrará uma coleção de arquivos em linguagem C que servem como exercícios para aprender os fundamentos da linguagem. Estes arquivos foram criados como parte do curso técnico de mecatrônica no Instituto Federal, abordando conceitos básicos, lógica de programação e cálculos simples.

## Conteúdo

Dentro deste projeto, você encontrará vários arquivos em linguagem C que abrangem:

- **Lógica de Programação**: Exemplos e exercícios para desenvolver suas habilidades de resolução de problemas.

- **Cálculos Simples**: Demonstração de como realizar cálculos matemáticos em C.

- **Fundamentos da Linguagem C**: Introdução aos conceitos básicos, como variáveis, estruturas condicionais e loops.

## Como Executar

Se você deseja praticar ou entender melhor os conceitos da linguagem C, siga estas etapas:

1. Certifique-se de ter um compilador C (como GCC) instalado em seu ambiente.

2. Clone este repositório para sua máquina local.

3. Navegue até a pasta deste projeto.

4. Compile e execute os arquivos C para ver os exemplos e exercícios em ação.

```bash
gcc arquivo.c -o programa
./programa
```
