Bem-vindo ao TechThrowback! Este repositório é uma coleção de projetos de iniciante que representam minha jornada de aprendizado na programação. A cada projeto, você encontrará uma visão do meu progresso e da evolução das minhas habilidades ao longo do tempo.

## Descrição

Neste repositório, você encontrará uma variedade de projetos que abrangem diversas tecnologias e conceitos, desde os meus primeiros passos na programação até o meu crescimento contínuo como desenvolvedor. Cada projeto tem seu próprio README individual, onde você pode encontrar detalhes, instruções de uso e muito mais.

## Projetos

Aqui estão os projetos disponíveis neste repositório:

1. [**Ruby Básico - #TechThrowback**](Projetos/Ruby): Você encontrará uma série de arquivos Ruby que abordam conceitos básicos da linguagem que explora tópicos fundamentais, como estruturas condicionais, loops, coleções e muito mais.
2. [**Linguagem C - #TechThrowback**](Projetos/Linguagem-C): Você encontrará uma coleção de arquivos em linguagem C que servem como exercícios para aprender os fundamentos da linguagem. Estes arquivos foram criados como parte do curso técnico de mecatrônica no Instituto Federal, abordando conceitos básicos, lógica de programação e cálculos simples.
3. [**Perguntas e Respostas - #TechThrowback**](Projetos/PerguntasRespostas): Este projeto específico é um site de perguntas e respostas, desenvolvido para ajudar a entender os conceitos de desenvolvimento web, tanto no frontend quanto no backend.

## Como Contribuir

Este repositório está em constante evolução, e novos projetos serão adicionados à medida que eu continuar a relembrar meus dias de aprendizado. Se você quiser contribuir com seu próprio projeto de iniciante ou fazer sugestões, sinta-se à vontade para abrir uma issue ou um pull request.

## Agradecimentos

Agradeço por fazer parte desta jornada de aprendizado e por explorar meus primeiros passos na programação. Espero que esses projetos possam ser úteis ou inspiradores para outros iniciantes!
