## Descrição

O projeto DevLinks faz parte do repositório TechThrowback e é uma aplicação web de listagem de links importantes, semelhante ao Linktree. Desenvolvido com HTML, CSS e Javascript puro, este projeto visa explorar conceitos de animação, responsividade, temas light e dark, além de diferentes tipos de displays no CSS.

## Funcionalidades

- **Listagem de Links**: Apresenta uma lista de links importantes para o usuário.

- **Tema Light e Dark**: Oferece suporte para temas claro e escuro, proporcionando uma experiência personalizada ao usuário.

- **Responsividade**: Garante uma experiência de usuário consistente em dispositivos de diferentes tamanhos.

## Como Rodar o Projeto

1. Clone este repositório para sua máquina local

2. Acesse o diretório do projeto:

    ```bash
    cd techthrowback-devlinks
    ```

3. Abra o arquivo `index.html` em seu navegador ou utilize uma extensão **Live Server** se estiver usando VSCode.
