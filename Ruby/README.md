## Descrição

Este é o primeiro projeto do repositório TechThrowback. Aqui, você encontrará uma série de arquivos Ruby que abordam conceitos básicos da linguagem. Este projeto é uma reminiscência dos meus primeiros passos na programação e explora tópicos fundamentais, como estruturas condicionais, loops, coleções e muito mais.

## Conteúdo

Dentro deste projeto, você encontrará os seguintes arquivos Ruby, cada um com seu próprio conjunto de conceitos:

1. **case_when_else.rb**: Demonstração de como usar as instruções `case`, `when` e `else` para tomar decisões condicionais em seu código.

2. **collection_array_hash.rb**: Exploração de coleções de dados em Ruby, incluindo arrays e hashes.

3. **conditions.rb**: Exemplos de uso de estruturas condicionais, como `if`, `elsif` e `else`.

4. **object_orientation.rb**: Introdução aos princípios da programação orientada a objetos em Ruby.

5. **request.rb**: Um exemplo de como fazer solicitações de rede usando a biblioteca `net/http`.

6. **variables.rb**: Demonstração de como declarar e usar variáveis em Ruby.

7. **while_for_each.rb**: Exemplos de loops, incluindo `while`, `for` e iteração com `each`.

## Como Executar

Se você quiser experimentar os códigos, siga estas etapas:

1. Certifique-se de ter Ruby instalado em seu ambiente.

2. Clone este repositório para sua máquina local.

3. Navegue até a pasta deste projeto.

4. Execute os arquivos Ruby para ver os exemplos em ação.

```bash
ruby case_when_else.rb
ruby collection_array_hash.rb
ruby conditions.rb
ruby object_orientation.rb
ruby request.rb
ruby variables.rb
ruby while_for_each.rb
```
